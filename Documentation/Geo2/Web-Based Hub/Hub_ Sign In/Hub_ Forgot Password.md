# Hub: Forgot Password

If you have forgotten or have not set up yet your password and cannot sign in successfully, follow the link `Forgot password?` on [**Sign in**](https://hub.geo2.com/en-GB/auth/signin) page.Â  This will enable you to reset your Geo2 password.

![Screenshot 2025-06-05 at 15.08.50.png](../../../attachments/776a6c01-e2af-4688-b6b8-2729ade55263.png)

After you have filled in your email address, press `Get code`. You will see a prompt for you to check your inbox for a verification code sent to the provided email address. You need to copy the code and paste it to the form in Hub:

![Screenshot 2025-06-05 at 15.11.19.png](../../../attachments/00f825a4-a82f-4dcf-95c1-862f27fbd8d4.png)![Screenshot 2025-06-05 at 15.11.03.png](../../../attachments/b7fdce4b-89e4-4d04-bd0f-0de085bffefc.png)

If you have not received an email with a verification code, press the `Resend email` button, and a new code will be sent. Once the code is pasted, press the `Continue` button. You will be redirected to create a new password.

![Screenshot 2025-06-05 at 15.12.35.png](../../../attachments/0165c255-9886-4920-819a-971f6d14c85f.png)

Press `Save` to create a new password. Once the password is changed, you will be logged in and redirected to Analytics page. A confirmation will appear:

![](../../../attachments/c3d3859b-82ed-4b16-8db3-8e7b000a4e98.png)
