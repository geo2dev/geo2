# Mobile App: Barcode Scanning

The mobile app is able to scan a wide range of barcode formats as follows:

|  **Format**                   |  **Android**    |  **iOS**    |
|:------------------------------|:----------------|:------------|
| Code 128                      | Y               | Y           |
| Code 39                       | Y (not mod43)   | Y           |
| Code 93                       | Y               | Y           |
| Codabar                       | Y               | Y           |
| EAN-13                        | Y               | Y           |
| EAN-8                         | Y               | Y           |
| ITF, more than 6 characters   | Y               | Y           |
| UPC-A                         | Y               | N           |
| UPC-E                         | Y               | N           |
| QR Code                       | Y               | Y           |
| PDF417                        | Y               | Y           |
| Aztec                         | Y               | Y           |
| Data Matrix                   | Y               | Y           |
| Single-character 1-D          | N               | N           |
| Encoded with FNC2, FNC3, FNC4 | N               | N           |
| QR Code in ECI mode           | N               | N           |
