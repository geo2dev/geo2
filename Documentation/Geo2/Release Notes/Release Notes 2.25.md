# Release Notes 2.25

# Release Date

2024-06-27

Bug fixes and refactoring. 

# New Features

|  **Key**                        |  **Summary**                                                                                                             |  **Description**                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                    |
|:--------------------------------|:-------------------------------------------------------------------------------------------------------------------------|:----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| IBALDEV-4511                    | Web UI: Created POD - product level failures when products are not added to packages                                     | In the mobile app, you can record failures at the product level on the Create POD page. Track these failures on the POD tab in the Hub within consignments or by pressing the POD icon on the consignment card. A list of products will be displayed, showing the product code, the successful quantity out of the total planned, status, and a failure reason code. These updates are also reflected on the public POD page and in the PDF, allowing consignees to track product failures as well. This feature helps enhance transparency and accountability by providing detailed information about product-level failures. It is useful for identifying and addressing issues promptly, improving overall efficiency and customer satisfaction. |
| IBALDEV-3372                    | Web UI: Highlight required fields with asterisk ( \* )                                                                   | Added asterisks ( \* ) near the titles of the required fields to identify which information is mandatory to fill in, and which is optional.                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                         |
| IBALDEV-5133 <br/> IBALDEV-5134 | Web UI: Environment settings - import depots <br/> Web UI: Add asterisks and text tooltips for columns for depot imports | In the Depots tab under environment settings, added the possibility import depot data. This feature is particularly beneficial if you have a large number of depots and want to save time adding them to the Hub. To import depots, click the "Import" button to open an import spreadsheet. You can enter depot data manually, copy and paste from your Excel file, or upload your Excel file by clicking the upload icon in the spreadsheet. Required fields are marked with an asterisk (\*) near the column titles. For added convenience, hover over each column title for text help to understand its purpose.                                                                                                                                |
| IBALDEV-5194                    | Web UI: Hide columns in Environment settings -> Vehicles                                                                 | Hide weight capacity, volume capacity, emission factor, and description columns on the Vehicles tab in the environment settings to save space. These fields are still available when you click the Edit button and open the dialog. The following columns are displayed in the table:  <br/><ul><li><p><span class="legacy-color-text-default">Key </span></p></li><li><p><span class="legacy-color-text-default">Registration</span></p></li><li><p><span class="legacy-color-text-default">Speed factor</span></p></li><li><p><span class="legacy-color-text-default">Depot</span></p></li><li><p><span class="legacy-color-text-default">Load building </span></p></li></ul>                                                                     |
| IBALDEV-4999                    | Web UI: Add consignments filtering by Depot on "All" tab                                                                 | Added the option to filter consignments by Depot on the "All" tab similar to "Last 7/30 days" with the selector field. It's possible to select any depot from the environment settings to narrow down the list of consignments.                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                     |
