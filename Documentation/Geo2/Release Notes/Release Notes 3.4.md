# New Features

|  **Key**     |  **Summary**                                                       |  **Description**                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                               |
|:-------------|:-------------------------------------------------------------------|:---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| IBALDEV-7170 | Hub/Mobile app: Integrate solution for more accurate delivery ETAs | **Planned timings (ETAs)** across stops in your routes **are now calculated with greater precision**, giving you more reliable results and helping you plan with confidence. Thanks to optimized algorithms for determining arrival times at stops with a new traffic consideration model, your routes reflect real-world conditions more accurately. These improvements are applied automatically when running **route calibration** or **optimization without time windows**, so you’ll benefit immediately without any extra steps. <br/> Learn more about [Hub: Routes](../Web-Based%20Hub/Hub_%20Routes.md) and [Hub: Routes](../Web-Based%20Hub/Hub_%20Routes.md).                                                                                                                                                                                                                                                                                       |
| IBALDEV-7127 | Hub: Implement new tool for selecting stops on the map             | You now have a more reliable way to **select stops directly on the map in both Route view and Map view**. At the bottom of the map, you’ll find several selection tools. By default, the **hand** tool is active. You can select stops individually by holding Ctrl/Command and clicking on them. <br/> To select multiple stops at once, choose one of the shape tools: <br/><ul><li><p><strong>Polyline</strong> – Place dots on the map to draw a custom shape. To complete the selection, finish by clicking back on your starting point.</p></li><li><p><strong>Square</strong> – Click once to set the starting corner, then click again when the square covers the desired stops.</p></li><li><p><strong>Circle</strong> – Works the same way as the square tool but with a circular selection.</p></li><li><p><strong>Free drawing</strong> – Use this tool to draw any shape freely, and all stops within the outline will be selected.</p></li></ul> |
