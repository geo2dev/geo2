| IBALDEV-6651 | Hub: Move Orders, Routes, Vehicle checks, etc. into vertical menu. Move organization and environment settings into one single page. | The header options like Orders, Routes, Vehicle checks, etc. have been moved to a new vertical menu for easier navigation, showing icons by default and expanding on hover. At the top of the menu, the organization and environment selector allows quick switching and creation based on user permissions. Settings for organizations and environments are now available in a single page with visibility and editing options depending on access levels. Updated layouts make it easier to find and manage key settings. <br/> Learn more about [Hub: Organization Settings](../Web-Based%20Hub/Hub_%20Organization%20Settings.md) and [Hub: Environment Settings](../Web-Based%20Hub/Hub_%20Environment%20Settings.md). |
| IBALDEV-6804 | Hub: Add {CONTACT\_NAME} token to subject of public tracking and POD email notifications                                            | You can now use the {CONTACT\_NAME} token in the subject line of public tracking and POD emails. A tooltip has been added to explain this token in both email settings. <br/> Learn more about [Hub: Environment Settings](../Web-Based%20Hub/Hub_%20Environment%20Settings.md) and [Hub: Environment Settings](../Web-Based%20Hub/Hub_%20Environment%20Settings.md).                                                                                                                                                                                                                                                                                                                                                       |
| IBALDEV-6764 | Hub: Default template for public tracking SMS                                                                                       | A default template for a public tracking SMS is now automatically added when a new account, organization, and environment are created during onboarding. The template includes key tokens like <CONTACT\_NAME>, <ORDER\_KEY>, and <PUBLIC\_TRACKING> to personalize tracking messages. <br/> Learn more about [Hub: Environment Settings](../Web-Based%20Hub/Hub_%20Environment%20Settings.md).                                                                                                                                                                                                                                                                                                                             |
| IBALDEV-4779 | Mobile app: Address scanning for route start/end points and recipient addresses                                                     | You can now **capture addresses instantly using your device’s camera**, no need for manual typing. Once scanned, the detected address appears for confirmation, with options to rescan or search for matching results. This feature speeds up entering route starts, route ends, and recipient addresses, reduces errors, and makes planning deliveries more efficient and hassle-free.                                                                                                                                                                                                                                                                                                                                     |
| IBALDEV-4782 | Mobile app: Voice search for route start/end points and consignee addresses                                                         | You can now **speak recipient or route addresses directly in the app**, with real-time speech recognition that quickly converts your voice into text. Easily select your preferred language, review the recognized address, and confirm it with “Search” or retry with “Speak again.” This hands-free feature saves time, reduces typing errors, and makes entering addresses more convenient, especially when managing multiple orders or routes.                                                                                                                                                                                                                                                                          |
| IBALDEV-7088 | Mobile app: "Add order" button between stop list on Route view page                                                                 | A new **“Add order” button** helps you quickly start or continue planning routes. When no orders exist, it appears between the route start and end cards with a green plus icon, and once orders are added, it moves to the end of the list for adding the next one. By pressing it, you will be redirected to Add order page to add an order to your route. <br/> Learn more about [Mobile App: Routes and Stops](../Mobile%20App/Mobile%20App_%20Routes%20and%20Stops.md).                                                                                                                                                                                                                                                |
| IBALDEV-7087 | Mobile app: Search bar with scan and voice search options on Route view page                                                        | A new **Search bar with scan and voice search** is available above the list of stops for users with order creation permissions. It allows entering an address manually, using geolocation, scanning, or speaking an address, which is then saved to the Recipient address field and used to quickly create a new order.                                                                                                                                                                                                                                                                                                                                                                                                     |
| IBALDEV-6937 | Mobile app: Editing of stop duration inside order                                                                                   | A new Stop duration field has been added to Add/Edit order pages, displayed below the Required to field and pre-filled with the default value from Environment settings. You can adjust it by typing in any value in minutes.                                                                                                                                                                                                                                                                                                                                                                                                                                                                                               |
| IBALDEV-6773 | Mobile app: Adding packages to orders on Add/Edit order pages                                                                       | You can now add packages to orders directly on Add order page, either by entering barcodes manually or scanning them with your device’s camera. Multiple barcodes can be added, scanned, and managed in a single list, giving you the ability to review, delete, or update them before saving.                                                                                                                                                                                                                                                                                                                                                                                                                              |
| IBALDEV-4775 | Mobile app: Displaying user's location on map on Route view page                                                                    | When location tracking is permitted and enabled in the app, your **current location** is always shown on Route view map with a **blue icon**.                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                               |
| IBALDEV-6707 | Mobile app: Upgrade from monthly to yearly subscription                                                                             | A new option allows you on a monthly Pro or Advanced subscription to upgrade to a **yearly plan with 20% savings**.                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                         |
| IBALDEV-6542 | Mobile app: Editing of profile info                                                                                                 | The Profile page now supports editing Full name, First name, and Last name fields with an Update button to save changes.                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                    |
