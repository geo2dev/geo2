# Release Notes 2.30

# Release Date

2024-11-01

Bug fixes. 

# New Features

|  **Key**                        |  **Summary**                                                                                                 |  **Description**                                                                                                                                                                                                                                                                                                                                                                                                                   |
|:--------------------------------|:-------------------------------------------------------------------------------------------------------------|:-----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| IBALDEV-5822                    | Web UI: Allow editing load properties on Load plan page                                                      | Added the ability to modify load properties such as user, vehicle, area, etc. right from the Load Plan page. For it, press the pencil icon by hovering over the field, and select a value you need. To save changes, simply click the tick icon on the field or anywhere on the page. Changing fields like depot, vehicle, or planned start will trigger automatic recalibration, updating route distance and timings accordingly. |
| IBALDEV-5782                    | Web UI: Possibility to copy consignment and load keys                                                        | Added to the ability to easily copy keys for consignments, loads, and stops. Simply hover over a key on any consignment, load, or stop card to see a tooltip displaying the full key along with a copy icon. Click the icon to copy the key, allowing you to paste it wherever needed.                                                                                                                                             |
| IBALDEV-5732 <br/> IBALDEV-5733 | Web UI/API: Add package name (description) for package entity                                                | Added the new field "Description" for a package entity inside a consignment. You can use it, for example, to describe its appearance so a driver can easily find it inside a vehicle or provide additional instructions.                                                                                                                                                                                                           |
| IBALDEV-5757 <br/> IBALDEV-5758 | Web UI/API: Restrict possibility to add both packages and products separately when user started linking them | It is still possible to add products and packages separately to a consignment. They will be displayed in separate sections on the Stop details and Create POD pages in the mobile app. However, if you started linking them (adding products inside of packages), you would need to link all the product quantities to packages.                                                                                                   |
| IBALDEV-5716                    | Web UI: Environment settings -> Users -> Add - possibility to select all users                               | When inviting users to your environment (Environment settings → Users → Add), provided the possibility to add all users at once to your environment, additionally to selecting each user one by one.                                                                                                                                                                                                                               |
| IBALDEV-5798                    | Web UI: Add password hints                                                                                   | For your comfort usage, added password hints (at least 8 characters, 1 uppercase and 1 lowercase, 1 number, and 1 special character) to each password field on the Registration, Sign in, and Change password pages.                                                                                                                                                                                                               |
| IBALDEV-5801                    | Web UI: Highlight Actions button in green on Load plan and Map view pages                                    | Highlighted the Actions button in green colour on the Load plan and Map view pages to make the main actions more visible on the pages.                                                                                                                                                                                                                                                                                             |
| IBALDEV-5925                    | Web UI: Make asterisks ( \* ) red for required fields                                                        | Highlighted stars near labels of the required fields in red colour to make them more visible on the pages.                                                                                                                                                                                                                                                                                                                         |
