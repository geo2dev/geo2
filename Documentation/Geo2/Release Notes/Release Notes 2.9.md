# Release Notes 2.9

# Release Date

2023-10-19

# New Features

|  **Key**     |  **Summary**                                                         |  **Description**                                                                                                                                                                                                                                                                                                                                                                              |
|:-------------|:---------------------------------------------------------------------|:----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| IBALDEV-3725 | Web UI: Message log                                                  | Added the possibility to track the status of POD emails (both successful and failed), public tracking emails, and SMS notifications. You will be able to see the date and time sent, the recipient, and the notification status. <br/> Statuses for emails are Sent, Undelivered, Delivered, Opened, and Clicked.  <br/> Statuses for SMS notifications are Sent, Undelivered, and Delivered. |
| IBALDEV-3829 | Web UI: Peak Vehicle Volume Capacity chart                           | Added the Peak Vehicle Volume Capacity chart to the Dashboard page. To calculate the peak used vehicle volume capacity, we take into account the average peak volume of all loads by vehicle. <br/> The filtering is by the load planned date.                                                                                                                                                |
| IBALDEV-3826 | Web UI: Peak Vehicle Weight Capacity chart                           | Added the Peak Vehicle Weight Capacity chart to the Dashboard page. To calculate the peak used vehicle weight capacity, we take into account the average peak weight of all loads by vehicle. <br/> The filtering is by the load planned date.                                                                                                                                                |
| IBALDEV-3883 | Web UI: Scrolling on the right side of the charts                    | Moved the internal scrolling on the charts to the more intuitive right side.                                                                                                                                                                                                                                                                                                                  |
| IBALDEV-3982 | Web UI: Change default value for Time at stop field                  | Changed the default value for the Time at stop field on the Loads tab in the Environment settings to 5 minutes instead of 4 to make the value rounded. It will be not applied to old environments.                                                                                                                                                                                            |
| IBALDEV-3985 | Web UI: Make options required for dropdown custom field              | Added the validation for a dropdown custom field in the Environment settings. Now it's only possible to create a dropdown custom field with at least 1 option provided.                                                                                                                                                                                                                       |
| IBALDEV-3502 | Mobile app: Calibrate button is enabled for single-consignment loads | Made it possible to calibrate a load in the mobile app, even when it only has one consignment.                                                                                                                                                                                                                                                                                                |
