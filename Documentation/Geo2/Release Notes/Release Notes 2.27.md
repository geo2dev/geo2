| IBALDEV-5361                                                                                                                   | Web UI: Address auto-completion                                                                                                                                                                                                                                                                                                                       | Made the Address line 1 field as an address search in the address form. Start typing the desired address, and results will appear. The selected address will automatically populate the address fields, which can be edited if needed. This feature is implemented for consignee and consignor addresses in consignments, as well as organization, environment, and depot addresses. This functionality speeds up the process of entering addresses, reducing manual input and the likelihood of errors. This is particularly useful for managing multiple consignments, as it significantly shortens the time required to create and update consignment details.                                                                                                                                                                                                                                                                                           |
| IBALDEV-5245 <br/> IBALDEV-5249 <br/> IBALDEV-5250 <br/> IBALDEV-5252 <br/> IBALDEV-5253 <br/> IBALDEV-5256 <br/> IBALDEV-5257 | Web UI: User guide for Organization/Environment settings <br/> Web UI: User guide for Consignments page <br/> Web UI: User guide for Loads page <br/> Web UI: User guide on Vehicle checks page <br/> Web UI: User guide on Load plan page <br/> Web UI: User guide for Dashboard page <br/> Web UI: "Start tour" button in header on different pages | Added the text guides for all pages in the Hub. When you visit a page for the first time, an automatic text guide will appear, explaining the main actions you can perform. These guides are available on the Dashboard, Consignments, Loads, Vehicle Checks, Organization, and Environment settings pages. If you prefer not to see the guide, you can click "Skip" on the tooltip to bypass the tour for that page, or "Skip tutorials on all pages" to disable guides on all pages. Later, if you want to revisit these guides, you can click the Question icon in the header toolbar and select the "Start tour" option to learn more about each page. These guides are designed to help you quickly understand and navigate the features of each page, enhancing your ability to utilize the system efficiently whether you're managing consignments, tracking loads, conducting vehicle checks, or configuring organization and environment settings. |
