| IBALDEV-7427 <br/> IBALDEV-7428 | Hub: Add "Stop card layout" settings to Settings -> Routes <br/> Hub: Apply "Stop card layout" settings for displaying info on stop cards on Route view/Map view pages and preview in Settings | The new **configurable stop card layout** lets you choose exactly what information appears on stop cards on Route and Map view pages. In **Settings → Routes**, a new **“Stop card layout”** section allows you to organize stop details using two lists — **Available fields** and **Displayed fields** — which can be managed by drag-and-drop or clicking. You can preview how cards look in collapsed or expanded views for both medium and large screens, with limits on how many fields can be displayed. Longer fields like Recipient address or Delivery instructions count as two fields and take up extra space in a collapsed view. You can configure layouts separately for each screen size and view, and the system will automatically display the right version for your device. |
| IBALDEV-7251                    | Hub: User invitation using mobile number                                                                                                                                                       | The “User invitation using mobile number” feature allows organizations to invite users by either email or mobile number. Admins can select the preferred method in the “Invite user” dialog, with fields and defaults adjusting accordingly. Invitations sent to mobile numbers can be accepted via Hub or the mobile app, but no activation SMS with a link is sent.                                                                                                                                                                                                                                                                                                                                                                                                                           |
