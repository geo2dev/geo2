| IBALDEV-5107 | Load building: Live log feed to web UI | Made changes to the load building process. Once consignments and load building parameters are selected, you are redirected to the Load building page where you can check the status of the load building process. At the top of the page, you will see the summary with the number of consignments, depots, and vehicles sent for the load building, who pressed the Generate button, and the date when it happened. Below, you can check the selected load building parameters. <br/> On the right side of the page, you can see the live logs notifying you about the status of the load building. On the left side, once the load building is completed, you can find the list of created loads with the number of consignments in it, assigned depot, and vehicle. By pressing the load key on one of the created loads, the selected load will be opened in the new tab of the browser. |
